# GLP-1 Claim Teardown — Workflow Template

## Per-clinic deliverable (20 min each)

### 1. Find the page
- Target: med-spa/GLP-1 clinic in same city/metro as a known warning-letter recipient
- URL: landing page for compounded semaglutide/tirzepatide/GLP-1 weight loss
- Confirm: page is actively advertising the treatment (not just informational)

### 2. Flag 3 phrases
Scan for these patterns (the exact ones FDA has been citing):

| Pattern | Example | FDA citation basis |
|---------|---------|-------------------|
| FDA-approval adjacent | "FDA-approved compounded semaglutide" | Compounded drugs cannot be FDA-approved |
| Brand equivalence | "Same as Ozempic/Wegovy/Mounjaro" | Implies identity with brand-name drug |
| Absolute outcomes | "Lose 20 lbs," "guaranteed results" | Unsubstantiated efficacy claims |
| Safety erasure | "No side effects" | Minimizes known risks |
| Clinical proof without source | "Clinically proven" without citation | Missing evidence proximity |

### 3. Write 3 rewrites
For each flagged phrase:
- Current: verbatim quote from page
- Issue: what makes it risky (1 sentence)
- Rewrite: safer version that preserves persuasive intent

### 4. Rewrite the hero section (1 better headline)
- Current H1: what the page says now
- Issue: why it's weak/dangerous
- Rewrite: safer, conversion-focused version
