**Founding customer incentive (first 5 files only, expires when claimed):**
$500 per Standard CaseFile, rate locked for 90 days from first order, in exchange for
(a) a written testimonial after the second file and (b) permission to reference the working
relationship (not case details) by organization type — or by name if separately approved in
writing. One founding slot per organization.

**Pricing rules (non-negotiable):**
1. Coverage checks answering `covered:false` are always free — never charge to say no.
2. No retroactive-coverage pricing exists at any price. If we weren't recording, there is no file.
3. The Declaration price never discounts. Discounting sworn custodian statements signals
   exactly the wrong thing to exactly this buyer.
